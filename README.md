



## Running the iOS App (FinanceApp)

Follow these steps to run the FinanceApp on your local machine using Xcode.


### 1. Clone the Repository
```bash
https://github.com/omidiora/FinanceApp-Youverify-Assessment
cd FinanceApp-Youverify-Assessment
```

## Open the Xcode Project

If you see a Finance.xcworkspace file (CocoaPods), open it:

open ios/Finance-YouVerify-Assement.xcworkspace


 Otherwise, open the .xcodeproj file:

open ios/Finance-YouVerify-Assement.xcodeproj

 ## Install Dependencies

If the project uses CocoaPods:

cd ios
pod install
cd ..
